# 🎮 AmethystClient v1.1 - Краткая памятка

## ⚡ Быстрый старт

### 📥 Установка
1. Распакуй `AmethystClient-v1.1-Source.zip`
2. Открой терминал в папке проекта
3. Выполни: `./gradlew setupDecompWorkspace` (первый раз)
4. Выполни: `./gradlew build`
5. Скопируй `build/libs/AmethystClient-1.1.jar` в `.minecraft/mods/`
6. Запусти Minecraft с Forge 1.8.9

---

## 🎯 Управление

| Клавиша | Действие |
|---------|----------|
| **RShift** | Открыть GUI |
| **ЛКМ** в GUI | Вкл/Выкл модуль |
| **Колесико мыши** на игрока | Добавить/удалить друга |
| **← →** в GUI | Сменить градиент |
| **Колесико** в GUI | Прокрутка |

## 📦 Модули

### ✅ Включены по умолчанию:
- **SoupCounter** - иконка супа + счетчик
- **ArrayList** - список модулей справа
- **ColorChanger** - градиенты
- **Nametag** - "СИЛЬНЕЙШИЙ ИГРОК" в F5

### ⚪ Выключены (включи в GUI):
- **HitDelayFix** - убирает задержку атаки

## 🐛 Проблемы?

### Мод не загружается
- Проверь версию Forge (1.8.9)
- Посмотри логи

### GUI не открывается
- Проверь, не конфликтует ли RShift с другими модами
- Попробуй переназначить клавишу в настройках

### Не работает HitDelayFix
- Убедись, что модуль включен в GUI
- На некоторых серверах может не работать из-за античита
