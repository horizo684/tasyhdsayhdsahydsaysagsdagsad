# 🎨 Визуальная схема AmethystClient v1.1

## 📺 Главный экран игры

```
╔═══════════════════════════════════════════════════════════════╗
║ AmethystClient                                                ║
║ 🥣 12                                     ╔═══════════════╗  ║
║                                           ║ Nametag      ║  ║
║                                           ║ SoupCounter  ║  ║
║                                           ║ HitDelayFix  ║  ║
║                                           ╚═══════════════╝  ║
║                                                               ║
║             ┌───────────────────────┐                        ║
║             │                       │                        ║
║             │   ИГРОВОЙ ЭКРАН       │                        ║
║             │                       │                        ║
║             │      [Персонаж]       │                        ║
║             │                       │                        ║
║             └───────────────────────┘                        ║
║                                                               ║
╚═══════════════════════════════════════════════════════════════╝
```

**Описание:**
- **Левый верх:** Название клиента (фиолетовый)
- **Под названием:** Иконка супа 🥣 + количество (12)
- **Правый верх:** ArrayList с активными модулями (цветной градиент)

**Управление:**
- **ЛКМ** на модуль - переключить ON/OFF
- **Стрелки ← →** - сменить градиент
- **Колесико** - прокрутка списка
- **ESC** - закрыть GUI

## 👁️ Режим F5 (вид от третьего лица)

```
╔═══════════════════════════════════════════════════════════════╗
║                                                               ║
║                     ┌─────────────────┐                      ║
║                     │ СИЛЬНЕЙШИЙ      │                      ║
║                     │    ИГРОК        │                      ║
║                     └─────────────────┘                      ║
║                           ↓                                   ║
║                        ╭─────╮                               ║
║                        │ 🧍 │  ← Твой персонаж              ║
║                        ╰─────╯                               ║
║                                                               ║
║                                                               ║
║            (Вид от третьего лица)                            ║
║                                                               ║
╚═══════════════════════════════════════════════════════════════╝
```

**Особенности:**
- Надпись "СИЛЬНЕЙШИЙ ИГРОК" с градиентом желтый→красный
- Темный фон под текстом
- Всегда повернут к камере
- Видно только в F5

## 👥 Добавление друга (колесико на игрока)

```
╔═══════════════════════════════════════════════════════════════╗
║                                                               ║
║                        [friend] Steve                         ║
║                           ╭─────╮                            ║
║         Ты →  🧍         │ 🧍 │  ← Друг                     ║
║               │          ╰─────╯                            ║
║               │                                               ║
║               ↓                                               ║
║      [Прицел на игроке]                                      ║
║      [Нажми колесико мыши]                                   ║
║                                                               ║
║  Chat: Added friend: Steve                                    ║
║                                                               ║
╚═══════════════════════════════════════════════════════════════╝
```

**Результат:**
- Возле ника появится **[friend]** зеленого цвета
- В чате сообщение об добавлении
- Повторное нажатие - удалит из друзей

## 📱 Soup Counter подробно

```
┌──────────────────┐
│ AmethystClient   │  ← Название клиента
├──────────────────┤
│ 🥣  12           │  ← Иконка супа + количество
└──────────────────┘

Вместо старого:
┌──────────────────┐
│ AmethystClient   │
│ Soups: 12        │  ← Текстовый вариант (старый)
└──────────────────┘
```

**Преимущества нового:**
- ✅ Красивая 3D иконка супа
- ✅ Компактнее
- ✅ Понятнее визуально
- ✅ Профессиональный вид
