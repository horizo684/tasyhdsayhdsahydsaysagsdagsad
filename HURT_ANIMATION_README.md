# Amethyst Client with No Hurt Cam

## Что изменено

В этой версии **встроенный NoHurtCam модуль был полностью заменён** на внешний мод **No Hurt Cam** от iLofiz.

### Что это значит?

- Старый код NoHurtCam был удалён из проекта
- В папку `mods/` добавлен файл `nohurtcam-1_0_0.jar`
- Этот мод использует Mixin для удаления тряски камеры при получении урона

### Как это работает?

При сборке проекта:
1. Создастся файл `AmethystClient-1.1.jar`
2. Папка `mods/` будет включена в сборку
3. При запуске Minecraft, No Hurt Cam автоматически загрузится вместе с AmethystClient

### Установка

После сборки проекта (./gradlew build):
1. Скопируйте `build/libs/AmethystClient-1.1.jar` в папку mods Minecraft
2. Скопируйте `mods/nohurtcam-1_0_0.jar` туда же (если он не был включён в сборку)

### Важно

No Hurt Cam работает автоматически и не требует настройки.
Он не будет отображаться в ClickGUI, так как это независимый мод с Mixin'ами.
